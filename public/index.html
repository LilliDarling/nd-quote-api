API Documentation

Endpoints

Get a random quote
GET /api/quotes/random

Example response:

{
  "status": "success",
  "data": {
    "text": "Normal is a dryer setting.",
    "author": "Anonymous",
    "tags": ["neurodiversity", "humor", "perspective"],
    "isPublished": true,
    "createdAt": "2023-05-15T12:34:56.789Z",
    "updatedAt": "2023-05-15T12:34:56.789Z"
  }
}
Get all quotes (paginated)
GET /api/quotes?page=1&limit=10
Get a specific quote
GET /api/quotes/:id

Authentication

Include your API key in the request headers:

{
  "X-API-Key": "your-api-key-here"
}

Code Example

// JavaScript example
async function fetchRandomQuote() {
  try {
    const response = await fetch('https://your-api-url.com/api/quotes/random', {
      headers: {
        'X-API-Key': 'your-api-key-here'
      }
    });
    
    const data = await response.json();
    return data.data;
  } catch (error) {
    console.error('Error fetching quote:', error);
  }
}
